Exploratory Data Analysis (EDA)

Analyzing customer churn and data preprocessing techniques

Project Overview

This project involves performing Exploratory Data Analysis (EDA) on customer churn data. The key steps include handling missing values, transforming variables, and extracting insights from data visualizations.

Key Steps:

  • Replacing blanks with 0 where tenure is missing and no total charges recorded.
  • Converting binary values (0 and 1) of Senior Citizen to Yes/No for better readability.
  • Visualizing customer churn distribution using pie charts.

Key Insights

Churn Rate

26.45% of customers have churned. Further analysis explores the reasons behind churn.

Data Preprocessing

Handled missing values, standardized categorical variables, and prepared the data for modeling.
